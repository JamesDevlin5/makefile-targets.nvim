*makefile-targets.txt*   Pick and run Makefile targets from inside Neovim

==============================================================================
INTRODUCTION                                        *makefile-targets*

makefile-targets.nvim lets you fuzzy-pick any target from your project's
Makefile and run it in a terminal split.

==============================================================================
SETUP                                               *makefile-targets-setup*

Call `setup()` somewhere in your Neovim config: >lua

  require("makefile-targets").setup({
    makefile_name   = "Makefile",
    finders         = { "lsp", "git", "buffer", "cwd" },
    desc_prefix     = "##",
    make_cmd        = "make",
    make_args       = "",
    exclude         = {},
  })
<

==============================================================================
COMMANDS                                            *makefile-targets-commands*

:MakefileTargets                                    *:MakefileTargets*
    Open the target picker. Available even without calling setup().

:MakefileTargetsRunLast                             *:MakefileTargetsRunLast*
    Re-run the last executed target. Notifies if no target has been run yet.

==============================================================================
CONFIGURATION                                       *makefile-targets-config*

All options and their defaults:

  keymap          "<Leader>m"               Normal-mode keymap for the picker.
                                            Set to `false` to skip registering
                                            one.

  dry_run_keymap  false                     Normal-mode keymap that opens the
                                            picker in dry run mode, running
                                            `make -n` instead of `make`.
                                            Set to `false` to disable. >lua
  require("makefile-targets").setup({
    dry_run_keymap = false,
  })
<

  telescope_keymap false                    Normal-mode keymap that opens the
                                            Telescope picker. Requires
                                            telescope.nvim. Set to `false` to
                                            disable. >lua
  require("makefile-targets").setup({
    telescope_keymap = "<Leader>mt",
  })
<
                                            Inside the Telescope picker:
                                            <CR>   run the selected target
                                            <C-d>  toggle dry run mode

  mini_keymap     false                     Normal-mode keymap that opens the
                                            Mini.pick picker. Requires
                                            mini.pick. Set to `false` to
                                            disable. >lua
  require("makefile-targets").setup({
    mini_keymap = "<Leader>mm",
  })
<
                                            Inside the Mini.pick picker:
                                            <CR>   run the selected target
                                            <C-d>  toggle dry run mode

  makefile_name   "Makefile"                Filename to search for, walked
                                            upward from the resolved root.

  finders         { "lsp", "git",           Ordered list of strategies used to
                    "buffer", "cwd" }        resolve the project root. Each is
                                            tried in turn; the first to return
                                            a path wins.

                                            Available finders:
                                            - "lsp"    root_dir of the first
                                                       active LSP client
                                            - "git"    git repository root
                                            - "buffer" directory of the
                                                       current file
                                            - "cwd"    Neovim's working dir

                                            Omit entries to disable finders,
                                            or reorder to change priority: >lua
  require("makefile-targets").setup({
    finders = { "git", "buffer" },
  })
<

  desc_prefix     "##"                      Comment prefix on the line directly
                                            above a target that is shown as its
                                            description in the picker: >makefile
  ## Build the project
  build: src/main.c
	gcc -o build src/main.c
<
                                            Change to match your convention: >lua
  require("makefile-targets").setup({
    desc_prefix = "#",
  })
<

  make_cmd        "make"                    The make executable to invoke.
                                            Change to use an alternative: >lua
  require("makefile-targets").setup({
    make_cmd = "gmake",
  })
<

  make_args       ""                        Extra arguments inserted between
                                            the executable and the target,
                                            applied to every run: >lua
  require("makefile-targets").setup({
    make_args = "-j4",
  })
<
                                            Override per-call via pick_target()
                                            (the dry_run_keymap uses this): >lua
  require("makefile-targets.core").pick_target({ make_args = "-n" })
<

  exclude         {}                        Target names to hide from the
                                            picker entirely: >lua
  require("makefile-targets").setup({
    exclude = { "clean", "distclean" },
  })
<
